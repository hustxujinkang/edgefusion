## 快速测试流程

### 步骤1：启动Modbus充电桩模拟器

首先启动模拟设备，这样可以在没有真实设备的情况下测试：

```bash
python modbus_charger_simulator.py
```

模拟器会监听 `localhost:502`，并模拟一个充电桩设备。

**模拟器寄存器说明**：
| 寄存器地址 | 描述 | 单位 | 说明 |
|------------|------|------|------|
| 0 | 电压 | V | 倍数10 (2200 = 220V) |
| 1 | 电流 | A | 倍数10 |
| 2 | 功率 | W | |
| 3 | 累计能量 | kWh | 倍数100 |
| 4 | 温度 | °C | 倍数10 |
| 5 | 状态 | - | 0=可用, 1=充电中, 2=故障 |

### 步骤2：启动EdgeFusion主程序

在另一个终端（保持模拟器运行）中：

```bash
# 确保已激活虚拟环境
conda activate edgefusion-env

# 启动主程序
python -m edgefusion.main
```

### 步骤3：访问Web监控界面

打开浏览器，访问：
```
http://localhost:5000
```

你会看到5个标签页：
1. **仪表板** - 系统总览
2. **添加设备** - 配置和添加新设备
3. **设备管理** - 管理已连接设备，读写寄存器
4. **控制策略** - 查看和执行策略
5. **API文档** - 完整的API接口说明

## Modbus模拟器使用

### 功能特性

- 真实模拟充电桩状态变化
- 支持外部写入寄存器
- 状态同步和日志输出
- 充电状态自动模拟

### 启动模拟器

```bash
python modbus_charger_simulator.py
```

可选参数：
- `--host`: 监听地址 (默认: 0.0.0.0)
- `--port`: 监听端口 (默认: 502)
- `--unit-id`: Modbus单元ID (默认: 1)

示例：
```bash
python modbus_charger_simulator.py --port 5020
```

### 模拟器行为

1. **空闲状态 (Available)**
   - 电压: ~220V
   - 电流: 0A
   - 功率: 0W
   - 温度: 25°C左右轻微波动

2. **充电状态 (Charging)**
   - 功率: 7-8kW波动
   - 电流: 随功率变化
   - 能量: 持续累积
   - 温度: 随功率上升

3. **故障状态 (Fault)**
   - 功率: 0W
   - 电流: 0A

### 通过寄存器控制

你可以通过写入寄存器来控制模拟器：

| 操作 | 寄存器 | 写入值 | 效果 |
|------|--------|--------|------|
| 开始充电 | 5 | 1 | 状态变为Charging |
| 停止充电 | 5 | 0 | 状态变为Available |
| 设置故障 | 5 | 2 | 状态变为Fault |
| 修改电压 | 0 | 数值 | 设置电压(x10) |

## Web界面操作

### 添加设备

1. 点击"**添加设备**"标签页
2. 填写设备配置：
   - **设备ID**: 例如 `charger_001`
   - **设备类型**: 选择"充电桩"
   - **主机地址**: `localhost`
   - **端口**: `502`
   - **单元ID**: `1`
3. 点击"**测试连接**" - 应该显示"连接成功"
4. 点击"**添加设备**"

### 设备管理

1. 点击"**设备管理**"标签页
2. 你会看到已添加的设备列表
3. 点击设备右侧的"**操作**"按钮

#### 读取寄存器

- 在"读取寄存器"区域输入寄存器地址
- 点击"读取"按钮
- 结果会显示在下方

#### 写入寄存器

- 在"写入寄存器"区域输入地址和值
- 点击"写入"按钮
- 结果会显示在下方

#### 快速访问

在"充电桩寄存器快速访问"表格中，直接点击"读取"按钮可以快速访问常用寄存器。

### 删除设备

在设备列表中点击"删除"按钮，确认后即可删除设备。

## 常见问题

### Q: 提示"连接失败"怎么办？

**A:** 
- 确保模拟器正在运行
- 检查主机地址和端口配置是否正确（默认localhost:502）
- 查看模拟器终端是否有错误信息

### Q: 写入寄存器后读出来还是旧值？

**A:** 这个问题已修复！请确保使用最新版本的模拟器（2026-02-24更新）。现在模拟器支持双向同步：
- 外部写入会同步到内部状态
- 有日志提示 `[同步] 外部写入...`
- 模拟器会基于新状态继续运行

### Q: 如何停止程序？

**A:** 在运行程序的终端按 `Ctrl+C` 停止。

### Q: 数据保存在哪里？

**A:** 
- 设备数据: `edgefusion.db` (SQLite数据库)
- 日志文件: `logs/` 目录（按日期分割）

### Q: Web界面无法访问？

**A:**
- 确认主程序已成功启动
- 检查端口5000是否被占用
- 尝试使用 `http://127.0.0.1:5000` 代替localhost

### Q: 如何查看API文档？

**A:** 在Web界面中点击"API文档"标签页，或者直接访问各个API端点。
